Reminiscence Collection

This project showcases a video game archive website that catalogues over 170 video games. It was inspired by my earlier work, the Reminiscence Collection, which is a poster and books detailing video game information. This website serves as my senior capstone project for Virginia Commonwealth University. It is designed to be responsive and accessible, featuring login and signup pages for enhanced user interaction.

Three-Letter Word Assigner

This project is a Three-Letter Word Assigner where you can can get a random word and have an option to listen through text-to-speech and you can change voice, pitch, rate, and volume, You can copy and paste the word too. You can also see the definition of the word, do a quiz, word playlist, and contact me if you want to. Also, the website change background based on time and it have accessibility options to change the appearance.

Crystal&Bites

This website is for a fictional restaurant named Crystal&Bites, which includes features like a menu, login/signup, about, and contact sections. I created it in the summer of 2022 as part of my initial exploration into web design, during a web design class. Eager to challenge myself, I aimed to complete this project within two weeks, focusing on learning about responsive website design.

Daydream Project

This website is crafted to display my daydreams, which I've represented through tangible items like objects, papers, and posters, reflecting thoughts I've pondered for about 10-30 minutes each. The site features images and thumbnails that offer a closer look at the details, accompanied by descriptions of the scenarios. Additionally, I've documented the creation process through photographs, including close-up shots, all of which are accessible on the second page of the daydream tab.

Documentation of Discuss&Connect: Where Communities Unite

In Fall 2023, for the Independent Studio class taught by VCU Graphic Arts Professor Roy McKelvey, I aimed to create a community forum. Initially, I considered coding it myself, but after discussing with Roy, I decided to make mockups using Adobe Photoshop. I experimented with different color schemes and settled on orange and blue. Weekly, I designed layouts for the homepage and discussion page, creating fictional posts and users for realism. In Spring 2024, I continued the project, following Roy's suggestion to use WordPress. I chose Siteground for hosting and a community forum template, adapting it to match my mockups while enhancing user experience.
